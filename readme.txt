BEFORE
INSTALL A MODULE CALLED "REQUESTS" (It allows the program to grab data from API)
Download Python3.
Setup Python path.
Install pip using cmd: python get-pip.py
Type in cmd: pip
If you get this error: 'pip' is not recognized as an internal or external command, operable program or batch file.
then you need to change pip environmental variables.

INSTALL requests. Type in cmd: pip install requests

RUN PROGRAM
To run the program, run the script on the GitHub repo.
